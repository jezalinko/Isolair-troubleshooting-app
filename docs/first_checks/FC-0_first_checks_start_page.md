# First Check — START

**Purpose:**  
Select the most appropriate First Check based on the current observed fault condition.

---

## Decision Step

### Step 1 — Current Issue
**Question:**  
What is the current issue with the Isolair Fire Tank?

Select the most relevant option:

- **Snorkel Pump Won’t Operate** → Proceed to **First Check — Snorkel Pump Won’t Operate**  
- **System Won’t Turn On (Hydraulic Pump Won’t Run)** → Proceed to **First Check — System Won’t Turn On**  
- **Hydraulic Pump Continues To Run** → Proceed to **First Check — Hydraulic Pump Continues To Run**  
- **Uncommanded Door Opening** → Proceed to **First Check — Uncommanded Door Opening**  
- **Leaking Doors** → Proceed to **First Check — Leaking Doors**

---

## Notes
- First Checks are intended for **rapid fault isolation** in the field.  
- If a First Check does not locate the fault, proceed to the relevant **through checks** for detailed troubleshooting.
