## Hydraulic Pump Output Pressure Adjustment

**System:** Hydraulic Pump Continues To Run  
**Source:** troubleshooting_guide.pdf (pp. 50–53)

---

## Purpose
Verify and, if required, adjust the Hydraulic Pump output pressure to ensure correct system operation and prevent component damage.

---

## Description

The Hydraulic Pump output pressure is the **maximum pressure** the pump will produce if it runs continuously.  
This is **not** the normal operating pressure of the hydraulic system.

- **Normal operating pressure:** Controlled by the Pressure Switch (850–1150 psi)
- **Pump output pressure:** Must be set to **1400–1500 psi**

> **NOTE**
>
> If the output pressure is set too low, the fire tank may not operate correctly.  
> If set too high, internal damage may occur if a Pressure Switch fails.

The output pressure is set during annual inspection but may decrease over time and should be checked after extended use (e.g. every **100 hours**).

---

## Step 1 — Checking Hydraulic Pump Output Pressure

**Question:**  
What is the Hydraulic Pump output pressure when commanded via the test function?

### Test Method
- Ensure the system is correctly configured and safe to operate.
- Position one person at the **Isolair Display Box**.
- Turn the system **ON**.
- Press the **Orange test button** on the **right-hand side of the Isolair Junction Box**.
- Observe the **Pressure Gauge** while the pump is running.

Refer to **Pic 29**.

---

### Result

- **Pressure above gauge limits**
  - Turn the system **OFF immediately**.
  - Proceed to **Step 2 — Setting Hydraulic Pump Output Pressure**.

- **Pressure between 1400–1500 psi**
  - Output pressure is correctly set.
  - No adjustment required.
  - Return the system to normal configuration.

- **Pressure between 1200–1400 psi**
  - Output pressure is marginal.
  - Adjustment is recommended.
  - Proceed to **Step 2 — Setting Hydraulic Pump Output Pressure**.

- **Pressure below 1200 psi**
  - Output pressure is too low.
  - Tank operation may be affected.
  - Proceed to **Step 2 — Setting Hydraulic Pump Output Pressure**.

---

## Step 2 — Setting Hydraulic Pump Output Pressure

> **CAUTION**
>
> Always keep a person stationed at the Isolair Display Box during adjustment.  
> If pressure rises excessively, the system must be shut down immediately to prevent damage.

### Adjustment Method
- Ensure the system is **OFF**.
- Remove the **Locking Cap** from the Pressure Adjusting Screw on the Hydraulic Power Pack.
- Identify the **Pressure Adjusting Screw**.

Refer to **Pic 31** and **Pic 32**.

---

### Adjustment Procedure
- With one person controlling power at the Isolair Display Box:
  - Turn the system **ON**.
  - Press the **Orange test button** on the Isolair Junction Box.
- While the pump is running:
  - Use a **large flat-blade screwdriver** to adjust the Pressure Adjusting Screw:
    - **Clockwise (IN):** Increases output pressure
    - **Anti-clockwise (OUT):** Decreases output pressure
- Monitor the Pressure Gauge continuously.

Refer to **Pic 33**.

---

### Completion
- Once output pressure is stable between **1400–1500 psi**:
  - Turn the system **OFF**.
  - Refit the **Locking Nut** and **Locking Cap** securely.
- Re-test system operation to confirm correct pressure behaviour.

---

## Next Action

- If output pressure is correct and the Hydraulic Pump continues to run →  
  Return to **Pressure Switch Check**.
- If output pressure cannot be set within limits →  
  Investigate Hydraulic Pump condition or internal system faults.
